This folder is created for tutorial of Justin Israel about writing plugins with python
For running this plugin in maya you should add this lines to maya.env in specific version directory of maya

MAYA_PLUG_IN_PATH = E:\Madoodia\_Python\_learning_python\plugins;
MAYA_SCRIPT_PATH = E:\Madoodia\_Python\_learning_python\plugins\mel;
PYTHONPATH = %PYTHONPATH%;E:\Madoodia\_Python\_learning_python\plugins\python;
